# Aguinaldo Generator (Vue 3 + Vite)

A simple app to generate a random "aguinaldo" amount.

## Features

- **Name Validation:** The app validates if a user is on the list of "inaanak" before allowing them to proceed to the gift selection.
- **Random Gift Amount:** Generates a random gift amount for the user.
- **GCash Integration:** Allows the user to input their GCash number to receive the gift.

## Setup

```bash
cd /Users/jglumbao/Desktop/code/Aguinaldo-generator
npm install
npm run dev
```
